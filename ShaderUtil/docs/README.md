# OptiX Toolkit ShaderUtil Library

The OptiX Toolkit ShaderUtil is a header-only library of commonly used utilities for OptiX and other raytracing programs. Among other things, the library provides utilities for

* [Vector math operations](/ShaderUtil/include/OptiXToolkit/ShaderUtil/vec_math.h)
* [Color representation and conversion](/ShaderUtil/include/OptiXToolkit/ShaderUtil/color.h)
* [Self-intersection avoidance for secondary rays](/ShaderUtil/docs/selfIntersectionAvoidance)
* [Ray cones for texture LOD control](/ShaderUtil/docs/rayCones)
* [Environment map sampling](/ShaderUtil/include/OptiXToolkit/ShaderUtil/CdfInversionTable.h)
* [Cubic texture filtering](/ShaderUtil/include/OptiXToolkit/ShaderUtil/CubicFiltering.h)
* [Stochastic texture filtering](/ShaderUtil/include/OptiXToolkit/ShaderUtil/stochastic_filtering.h)
* [Transform matrices](/ShaderUtil/include/OptiXToolkit/ShaderUtil/Transform4.h)
